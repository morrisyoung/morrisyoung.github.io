Softwares [top]

Fast IBD Extraction from Ancestral Recombination Graphs [code]
Identity by Descent (IBD) segments are well defined as the long haplotype sharing between two individuals in a cohort. There are lots of IBD related research works, both in theoretical aspect and applied aspect. In all of these works, a very important problem is that, we always need some ground true IBD segments as the reference. This work is focused on extracting these ground true IBD segments directly from ancestral recombination graphs (ARG), in an efficient manner. This work will become an helpful tool for other related research, like theory verification of IBD, simulation based approach for demographic inference.
